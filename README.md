This is a project of a C# library that contains a group of useful methods for ERP systems.

---------------------------------------------------------------------------------------------------------------------------------

Requirements:

- Visual Studio (2017 or above), available on https://visualstudio.microsoft.com/downloads/.
- .NET Framework 3.5 or above.
